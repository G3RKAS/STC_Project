# Тестовое задание для компании СТЦ

Необходимо реализовать: 
1. Класс, который в отдельном потоке (запускается и останавливается через API класса) 
читает байты из source (ввод, предпочтительно должны реализовывать интерфейс 
QIODevice класса), конвертирует их в строковые символы и передает результат 
конвертации в sink (вывод, предпочтительно должны реализовывать интерфейс QIODevice 
класса).  
Байт, прочитанный из source, состоит из битов type (6-ой и 7-ой), кодирующие тип 
данных, хранящиеся в данном байте, и битов data (0-5), хранящие данные:  
<ul>
  <li>Если type = 0b00, то data – беззнаковое целое;</li>
  <li>Если type = 0b01, то data – знаковое целое;</li>
  <li>Если type = 0b10, то data – одина из букв латинского алфавита (‘a’ = 0b000000, ‘b’ 
= 0b000001, ‘c’ = 0b000010, ‘d’ = 0b000011 и т.д.);</li>
</ul>

3. Интерфейсы source и sink; 

4. Тесты с использованием gtest, gmock и их сборку через CMake;
   
6. Документацию к коду.
   
Все, что не указано, отдается на усмотрение исполнителю
